# Test project for Edsson (backend)

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ npm run start:dev
```

Project will open [http://localhost:3001](http://localhost:3001)


## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
